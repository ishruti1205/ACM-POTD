## Problem Statement
[Easy] Pascal's Triangle

## Approach
I used an iterative approach to build Pascal’s Triangle row by row, where each element is computed as the sum of the two elements above it from the previous row, resulting in O(n²) time complexity.

## Code

```java
import java.util.*;

class Solution {
    public List<List<Integer>> generate(int numRows) {

        List<List<Integer>> triangle = new ArrayList<>();

        for (int i = 0; i < numRows; i++) {

            List<Integer> row = new ArrayList<>();

            for (int j = 0; j <= i; j++) {

                // First and last element
                if (j == 0 || j == i) {
                    row.add(1);
                } 
                else {
                    int val = triangle.get(i - 1).get(j - 1) 
                            + triangle.get(i - 1).get(j);
                    row.add(val);
                }
            }

            triangle.add(row);
        }

        return triangle;
    }
}
```
