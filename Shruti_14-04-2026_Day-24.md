## Problem Statement
[Easy] Find the Center of Star Graph

## Approach
I used the fact that in a star graph the center node appears in every edge, so by comparing the first two edges, the common node between them is identified as the center in O(1) time.

## Code

```java
class Solution {
    public int findCenter(int[][] edges) {
         int a = edges[0][0];
        int b = edges[0][1];

        int c = edges[1][0];
        int d = edges[1][1];

        if (a == c || a == d) return a;
        else return b;
    }
}
```
